# Примеры Уравнений / Equation Examples

## Примеры для калькулятора уравнений

### Линейные уравнения / Linear Equations

```
2*x + 5 = 11
3*x - 7 = 2
5*x + 10 = 0
x/2 + 3 = 7
```

### Квадратные уравнения / Quadratic Equations

```
x**2 - 4 = 0
x**2 + 3*x + 2 = 0
x**2 - 5*x + 6 = 0
2*x**2 + 3*x - 2 = 0
x**2 + 1 = 0
```

### Кубические и полиномиальные / Cubic and Polynomial Equations

```
x**3 - 2*x**2 + x = 0
x**3 - 6*x**2 + 11*x - 6 = 0
x**4 - 16 = 0
x**3 + x**2 - x - 1 = 0
```

### Логарифмические уравнения / Logarithmic Equations

```
log(x) = 3
log(x) + 2 = 5
log(x) - log(2) = 1
2*log(x) = 4
log(x**2) = 2
```

### Логарифмы по разным основаниям / Logarithms with Different Bases

```
log(x, 2) = 3
log(x, 10) = 2
log(x, 5) = 1
```

**Примечание:** `log(x, base)` - логарифм x по основанию base

### Экспоненциальные уравнения / Exponential Equations

```
exp(x) = 5
exp(x) - 3 = 0
2*exp(x) = 10
exp(x) + exp(-x) = 2
exp(2*x) = 4
```

### Тригонометрические уравнения / Trigonometric Equations

```
sin(x) = 0
sin(x) = 0.5
cos(x) = 1
cos(x) = 0
tan(x) = 1
sin(x)**2 = 0.25
```

### Смешанные уравнения / Mixed Equations

```
x**2 + sin(x) = 1
log(x) + x = 2
exp(x) - x**2 = 0
x*log(x) = 1
```

### Рациональные уравнения / Rational Equations

```
1/x = 2
1/x + 1 = 3
x/(x+1) = 2
(x+1)/(x-1) = 3
```

### Иррациональные уравнения / Irrational Equations

```
sqrt(x) = 4
sqrt(x) + 2 = 5
sqrt(x**2 + 1) = 2
x**0.5 = 3
```

## Советы по вводу / Input Tips

1. **Умножение / Multiplication:**
   - Используйте `*`: `2*x`, `x*y`
   - Неявное умножение также поддерживается: `2x` → `2*x`

2. **Степени / Powers:**
   - Используйте `**`: `x**2`, `x**3`
   - Или функцию: `pow(x, 2)`

3. **Корни / Roots:**
   - Квадратный корень: `sqrt(x)` или `x**0.5`
   - Кубический корень: `x**(1/3)`

4. **Логарифмы / Logarithms:**
   - Натуральный: `log(x)` или `ln(x)`
   - По основанию: `log(x, base)`
   - Десятичный: `log(x, 10)`

5. **Константы / Constants:**
   - π: `pi`
   - e: `E` или `exp(1)`

6. **Функции / Functions:**
   - sin, cos, tan
   - asin, acos, atan
   - sinh, cosh, tanh
   - exp, log, sqrt
   - abs

## Ограничения / Limitations

- Калькулятор ищет символьные и аналитические решения
- Некоторые трансцендентные уравнения могут иметь бесконечное множество решений - будут показаны основные
- Для некоторых сложных уравнений численное решение может быть недоступно

## Примеры с результатами / Examples with Results

### Пример 1: Квадратное уравнение
**Ввод:** `x**2 - 4 = 0`
**Решения:** x = -2, x = 2

### Пример 2: Логарифм
**Ввод:** `log(x) + 2 = 5`
**Решение:** x = e³ ≈ 20.085537

### Пример 3: Экспонента
**Ввод:** `exp(x) - 5 = 0`
**Решение:** x = ln(5) ≈ 1.609438

### Пример 4: Тригонометрия
**Ввод:** `sin(x) = 0.5`
**Решения:** x = π/6, x = 5π/6 (и другие периодические решения)

### Пример 5: Полином
**Ввод:** `x**3 - 2*x**2 + x = 0`
**Решения:** x = 0, x = 1
